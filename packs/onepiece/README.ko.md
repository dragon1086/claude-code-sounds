# 원피스 사운드팩

원피스 애니메이션의 실제 명장면과 명대사로 구성된 사운드팩.  
AI 생성 음성이 아닌 원본 애니메이션 음성입니다.

## 이벤트 → 캐릭터 → 장면

| 이벤트 | 캐릭터 | 장면 / 대사                                   |
|--------|--------|-------------------------------------------|
| `sessionstart` | 루피 | "나는 몽키D.루피, 해적왕이 될 사나이다!"                 |
| `setup` | 샹크스 | 패기 방출 — 압도적인 존재감                          |
| `pretooluse` | 조로 | 사자의노래 기술 시전 — 칼을 뽑는 순간                    |
| `posttooluse` | 프랑키 | "수퍼!"                                     |
| `posttoolusefailure` | 로빈 | "살고싶어!!"                                  |
| `userpromptsubmit` | 루피 | "고무고무의~!" — 기술 준비                         |
| `stop` | 프랑키 | "수퍼!"                                     |
| `stopfailure` | 루피 | 에이스를 부르는 절규 — 극한의 절망                      |
| `notification` | 프랑키 | "와우!"                                     |
| `permissionrequest` | 나미 | 울면서 "루피 도와줘!"                             |
| `permissiondenied` | 나미 / 상디 | 상디 나미상~~ 달려들다 → 나미 펀치                     |
| `sessionend` | 상디 | "더럽게 신세 많이 졌습니다, 은혜를 잊지 않겠습니다" (제프 선생 작별) |
| `subagentstart` | 조로 | "롤로노아 조로" — 자기소개                          |
| `subagentstop` | 조로 | 삼도류 오의·삼천세계 — 임무 완료                       |
| `taskcreated` | 우솝 | 우솝 망치 소리 — 새 작업 착수                        |
| `taskcompleted` | 조로 | 삼도류 오의·삼천세계                               |
| `teammateidle` | 조로 | "아무일도 없었다" — 피투성이로 희생하고 안아픈척              |
| `cwdchanged` | — | 원피스 애니 장면전환 인트로                           |
| `filechanged` | 우솝 | 우솝 망치 소리 — 파일 수정                          |
| `worktreecreate` | 루피 | 루피 웃음소리 — 새 모험 시작!                        |
| `worktreeremove` | 브룩 | 요호호호 — 작별                                 |
| `precompact` | 닥터 히루루크 | "사람이 언제 죽는다고 생각하나?"                       |
| `postcompact` | 로빈 | 로빈 웃음소리 — 상쾌한 재출발                         |
| `configchange` | 우솝 | "나는 원래부터 비관주의자다!!" — 상황이 반전됐다             |
| `elicitation` | 초퍼 | 귀여운 초퍼 부끄러움 — 조심스럽게 물어보는 중                |
| `elicitationresult` | 상디 | "멜로린~!" — 완전 홀려버림                         |
| `instructionsloaded` | 흰수염 | "원피스는 실재한다!!"                             |
